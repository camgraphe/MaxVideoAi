## Ce que vous cherchez
Le reference-to-video est plus large que le character reference. La référence peut être une image produit, une vidéo existante, une ambiance, une piste audio ou une direction de campagne. Le but est de guider le résultat, pas toujours de conserver une identité exacte.

## Meilleurs choix
1. **[Seedance 2.0](/fr/modeles/seedance-2-0)** - meilleur choix global pour workflows de références multimodales.
2. **[Veo 3.1](/fr/modeles/veo-3-1)** - très bon pour un plan propre à partir d’une référence image.
3. **[Kling 3 Pro](/fr/modeles/kling-3-pro)** - meilleur quand les références doivent devenir une séquence structurée.
4. **[LTX 2.3 Pro](/fr/modeles/ltx-2-3-pro)** - meilleur si le workflow continue avec extend, retake ou 4K.

## Pourquoi Seedance est premier
Seedance 2.0 est le meilleur point de départ quand vous arrivez avec du matériel de référence. Il convient aux équipes qui ont déjà des assets de campagne, des produits, des style frames ou des directions sonores.

Kling 3 Pro devient meilleur lorsque la référence doit rester cohérente sur plusieurs plans. LTX 2.3 Pro est le meilleur choix si vous devez ensuite corriger ou prolonger le résultat.

## FAQ
### Quelle différence entre image-to-video et reference-to-video ?
L’image-to-video anime souvent une seule image. Le reference-to-video utilise un ensemble plus large de références pour guider le rendu.
