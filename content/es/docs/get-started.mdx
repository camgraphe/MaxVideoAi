1. **Crea un workspace** e invita a tu equipo desde el panel. Elige si quieres un wallet compartido o saldos individuales para cada productor.
2. **Completa el brief de marca** para que los valores por defecto y las reglas de enrutamiento apliquen a todos. Paletas de color, overlays de logo y ratios preferidos se sincronizan para todos los engines.
3. **Conecta un wallet** o carga Starter Credits (10 $) para empezar. Puedes precargar con tarjeta, transferencia o factura; los reembolsos por renders fallidos son automáticos.

### Configurar roles

- **Productores**: pueden lanzar renders, gestionar jobs y recargar créditos.
- **Reviewers**: dejan comentarios frame a frame sin costo.
- **Finanzas**: ve gastos, facturas y previsiones de coste, pero no puede generar.

### Checklist del primer render

1. Ve a la pestaña **Examples** y clona un job reciente que encaje con tu caso de uso.
2. Cambia el prompt y los medios, luego revisa la etiqueta *Price Before You Generate* para confirmar el coste.
3. Haz clic en **Render** y sigue el estado en el feed. Cada run registra el engine, la duración y los retries para tu auditoría.

### Acceso API

¿Necesitas control programático? Solicita credenciales en support@maxvideo.ai y revisa la sección de webhooks abajo. La API REST refleja el workspace—mismos prompts, mismas reglas de enrutamiento—para automatizar campañas de alto volumen sin reescribir briefs para cada provider.
