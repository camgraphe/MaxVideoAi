## Qué estás buscando
Character reference es más estricto que una referencia visual normal. Pides conservar identidad, silueta, ropa, props y estilo aunque cambien cámara y acción.

## Mejores opciones
1. **[Happy Horse 1.0](/es/modelos/happy-horse-1-0)** - mejor opción general cuando las referencias de personaje también necesitan audio nativo, lip-sync y edición V2V.
2. **[Kling 3 Pro](/es/modelos/kling-3-pro)** - mejor para continuidad de personaje o prop con Elements, multi-shot y control de escena.
3. **[Seedance 2.0](/es/modelos/seedance-2-0)** - excelente cuando la identidad forma parte de un paquete de referencias más amplio.
4. **[Kling 3 Standard](/es/modelos/kling-3-standard)** - mejor opción value para probar continuidad.

## Por qué Happy Horse lidera
Happy Horse 1.0 lidera porque su modo R2V usa varias imágenes de referencia y queda junto a image-to-video, text-to-video y V2V. Es útil cuando el personaje también debe hablar, conservar audio nativo o recibir una edición visual posterior.
