## Qué estás buscando
Character reference es más estricto que una referencia visual normal. Pides conservar identidad, silueta, ropa, props y estilo aunque cambien cámara y acción.

## Mejores opciones
1. **[Kling 3 Pro](/es/modelos/kling-3-pro)** - mejor para continuidad de personaje o prop con Elements, multi-shot y control de escena.
2. **[Seedance 2.0](/es/modelos/seedance-2-0)** - excelente cuando la identidad forma parte de un paquete de referencias más amplio.
3. **[Happy Horse 1.0](/es/modelos/happy-horse-1-0)** - útil cuando las referencias de personaje también necesitan audio nativo, lip-sync y edición V2V.
4. **[Kling 3 Standard](/es/modelos/kling-3-standard)** - mejor opción value para probar continuidad.

## Por qué Kling 3 lidera
Kling 3 Pro lidera porque Elements y el control multi-shot dan una continuidad más robusta que repetir descripciones largas. Seedance 2.0 es fuerte cuando el personaje forma parte de un paquete de referencias más amplio. Happy Horse 1.0 sigue siendo útil para R2V, audio, lip-sync y V2V, pero no supera a Kling 3 Pro ni a Seedance en continuidad visual estricta.
