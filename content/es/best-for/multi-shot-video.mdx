## Qué estás buscando
El usuario no quiere un solo plano. Quiere una secuencia que parezca editada: apertura, detalle, acción, reveal y plano final. Técnicamente puede llamarse multi-shot, multi-prompt o prompt chaining.

## Mejores opciones
1. **[Kling 3 Pro](/es/modelos/kling-3-pro)** - mejor opción general para multi-shot, Elements, shot type y audio.
2. **[Seedance 2.0](/es/modelos/seedance-2-0)** - mejor si la secuencia también necesita referencias y audio nativo.
3. **[Sora 2 Pro](/es/modelos/sora-2-pro)** - útil para mini-secuencias genéricas y pulidas.
4. **[Kling 3 Standard](/es/modelos/kling-3-standard)** - mejor opción value para probar estructura.

## Por qué Kling 3 Pro lidera
Kling 3 Pro encaja mejor con esta intención: permite pensar en escenas, duración, Elements, shot type y continuidad. Seedance 2.0 es fuerte si referencias y audio pesan tanto como la estructura. Sora 2 Pro es útil para escenas genéricas, pero menos para referencias estrictas.
